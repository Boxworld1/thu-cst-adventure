# 專業課程



## 總覽

專業課分為兩大類：

* 必修課程：本系學生需要主部修讀完畢方可畢業；

* 選修課程 (自主發展課)：選擇多樣，可按照個人喜好選擇，惟選擇的課程方向需滿足培養方案所述之條件。

## 課程一覽

* 下述課程之修讀學期僅代表本人選擇對應課程之實際學期，並不構成任何選課建議；
* 強烈建議閣下根據自身的情況**合理分配時間**修讀課程，避免因為壓力過大導致身體健康出現問題，**生命無價**。
