## 心得

由於國際班課程進度會因應選課同學的水平不同而有所改變，因此這個學期有很多有用的內容（如 SVD）未能在課程中覆蓋。

相比起大陸班的課程（乃至社科國際班），國際班的考試可謂是十分簡單。考試內容主要為簡單計算（難度與功課相當），對概念的要求幾乎沒有，因此簡單計算的準確率在這門課中十分重要。
